Общие требования:
Этап 1. На основе заполненных анкетных данных заявителя на основании предоставленных шаблонов форм анкет в формате PDF с размеченными полями для редактирования – автоматически сформировать PDF файлы с заполненными полями.
1. Анкетные данные заявителя хранятся в файле с именем Clients_for_PDF формата Google Sheets на GoogleDisk. 
2. В одном файле может быть несколько заявителей. Каждый заявитель представлен отдельной вкладкой с названием, соответствующим его фамилии.
3. Имена полей представлены в колонке "B"
4. Значения полей представлены в колонке "A"
5.	PDF-шаблоны анкет хранятся в  папке PDF_templates на GoogleDisk. 
6. Каждый шаблон имеет свои обозначения полей ввода и следовательно требует своего маппинга (проставления соответствия между полями анкеты и полями шаблона).
7. Маппинг полей хранится во внешнем файле Mapping формата Google Sheets на GoogleDisk.  
8. Первая строка файла Mapping начиная с колонки "B" является названием шаблона
9. Колонка "A" содержит названия полей в файле анкет GoogleSheets.
10. Колонка "A" может содержать специальную строку "#DAY", означающий что значение поля анкеты с именем до специальной строки должно быть воспринято как дата и надо взять из нее только число месяца.
11.	Результирующие файлы (заполненные анкеты) в формате PDF с редактируемыми полями помещаются в  папку FilledPDF на GoogleDisk с созданием подпапки с именем заявителя. 
12. Имя заявителя берется из названия вкладки в файле Clients_for_PDF.
13. Число результирующих файлов будет равно числу шаблонов, т.к. на этом этапе мы не знаем кому какие шаблоны надо заполнять.
